## 手部追蹤
* 抓取掌心的21個點，將點連成線，追蹤手的位置。


## 偵測顏色
### 操作步驟
1. 將畫筆展示在螢幕中，利用控制條視窗抓取筆蓋的顏色。
2. 直至筆蓋對應到mask的視窗為白色，才算抓取成功。
